# Ledger — Personal Finance & Subscription Tracker

Track spending and recurring bills, in one calm place.

## Live Demo
- Frontend: https://ledger-finance-tracker.vercel.app
- Backend API: https://ledger-finance-tracker.onrender.com

## Tech Stack
- Frontend: React, Vite, React Router, Recharts, Framer Motion
- Backend: Node.js, Express, MongoDB (Mongoose), JWT Auth
- Deployment: Vercel (frontend), Render (backend), MongoDB Atlas (database)

## Features
- User authentication (JWT-based)
- Transaction tracking
- Subscription/recurring bill management with automated billing cron job

## Note
Backend is on Render's free tier — first request after inactivity may take 20-30 seconds to respond (cold start).
